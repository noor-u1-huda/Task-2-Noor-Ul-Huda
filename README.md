# DecodeLabs Project 2
## Data Classification Using AI

This project implements Iris Flower Classification using the K-Nearest Neighbors (KNN) algorithm in Python.

## Features
- Dataset Loading
- Feature Scaling
- Train-Test Split
- KNN Classification
- Accuracy Evaluation
- Confusion Matrix
- Classification Report
- Visualization

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib
